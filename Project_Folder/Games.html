Our Games

2048

2048 is a single-player sliding block puzzle game. Players combine numbered tiles on a 4x4 grid to reach the tile with the number 2048. The game ends when no more moves are possible.

Balloon Shooter

Balloon Shooter is an arcade game where players pop moving balloons to score points before time runs out.

Snake Game

The Snake game is a classic arcade where players control a growing snake, eating pellets to score points while avoiding collisions.

Word Search Game

Word Search is a puzzle game where players find hidden words in a grid of letters arranged in various directions. The goal is to locate all specified words from a list within the grid.

Pong Game

Pong is one of the earliest arcade video games, created by Atari and released in 1972. It is a table tennis sports game featuring simple two-dimensional graphics

Quiz Game

A quiz game is a type of interactive game where players are presented with questions and have to select the correct answers from a list of choices.

Break Out

Breakout is a classic arcade game where players use a paddle to bounce a ball, breaking bricks to score points. The goal is to clear all bricks on the screen without letting the ball fall off the paddle.

Tic_Tac_Toe

Tic Tac Toe is a two-player game where opponents aim to align three of their symbols (X or O) in a row to win.

Mines

Minesweeper is a classic logic-based computer game where players uncover squares on a grid without detonating hidden mines. The numbers revealed help players deduce the locations of mines, requiring careful strategy and logical thinking.

Guess The Word

Guess the Word is a challenging puzzle game where players deduce a hidden word by guessing letters within a limited number of attempts.

Sudoku

Sudoku is a logic-based number puzzle where players fill a 9x9 grid so that each column, row, and 3x3 subgrid contains all digits from 1 to 9. The game enhances problem-solving skills and offers varying levels of difficulty.

Maze Game

Maze is a puzzle game where players navigate through a complex network of paths or passages to find an exit. It challenges problem-solving skills and spatial awareness as players encounter dead ends and false routes.
